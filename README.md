# 📚 Havruta Study - FE

## Havruta?
친구를 의미하는 히브리어인 ‘하베르’에서 유래한 용어로, <br />
학생들끼리 짝을 이루어 서로 질문을 주고 받으며 논쟁하는 유대인의 전통적인 토론 교육 방법

### 핵심 방법론
1. 자신의 생각을 조직화하여 상대방에게 설명한다.
2. 상대방의 이야기를 듣고 질문한다.
